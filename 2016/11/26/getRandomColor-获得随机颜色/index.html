getRandomColor获得随机颜色

在此前完成的FCC弹幕APP项目中的JS代码有这样一段代码：

这段代码的作用是给变量getRandomColor赋值一个匿名函数，用于获取随机颜色。这段代码虽然简短，但仔细分析可以从中复习到如下知识点：

1. CSS中的颜色值

a.合法的颜色值

可以用以下方法来规定 CSS 中的颜色：

  • 十六进制色
  • RGB 颜色
  • RGBA 颜色
  • HSL 颜色
  • HSLA 颜色
  • 预定义/跨浏览器颜色名

b. 十六进制颜色值

所有浏览器都支持十六进制颜色值。

十六进制颜色是这样规定的：#RRGGBB，其中的 RR（红色）、GG（绿色）、BB（蓝色）十六进制整数规定了颜色的成分。所有值必须介于 0 与 FF 之间。

jQuery.css()方法中颜色值是必须是字符串形式，若想获得随机颜色，采用十六进制颜色实现起来最简单：将”#”和”RRGGBB”分开，仅需处理后面6位十六进制数。

2. Math.random()

a. 定义

Math.random() 函数返回 [0-1) 的浮点值伪随机数（大于等于0，小于1）。

此函数的随机数生成器和java一样以当前时间为随机数种子。

b. 用法举例

  • 返回一个大于等于0，小于1的伪随机数

  • 返回一个介于min和max之间的随机数

  • 返回一个介于min和max之间的整型随机数

3. 十六进制数（hex）

a. 定义

b. JS中不同进制数值的字面表示方法

  • 二进制：前缀为0b或0B，如0b11。
  • 八进制：前缀为0o或0O，或者前缀为0且只用到0-7的阿拉伯数字，如0037。
  • 十六进制：前缀为”0X”或”0x”，如0xff。

c. Javascript内部会自动将二进制、八进制、十六进制转换为十进制：

Math.random()*0x1000000 的结果为十进制小数，且必然在 [0, 16777216)范围之内。因 0x1000000 是7位十六进制数中最小的，可以确保最终计算得到的十六进制数值的位数不会超过6。

4. 左移运算符<<

左移运算符表示将一个数的二进制值向左移动指定的位数，尾部补0，即乘以2的指定次方（最高位即符号位不参与移动）。

位运算符只对整数起作用，如果一个运算子不是整数，会自动转为整数后再执行。

在getRandomColor函数中，<<0 用来将 Math.random() * 0x1000000 运算结果取整。同样，可以使用>>0来达到相同效果

5. Number.prototype.toString()

a.定义

toString() 方法返回指定 Number 对象的字符串表示形式。

b.语法

c.参数

radix:

指定要用于数字到字符串的转换的基数(从2到36)。如果未指定 radix 参数，则默认值为 10。

d.例子

在getRandomColor()中，Math.random() * 0x1000000 << 0 的结果为Number对象，使用toString()方法并指定radix为16即可得到对应的十六进制数值的字符串形式。

由上图可以看到出现了5位数的十六进制字符串，需要进行补位才能得到合法的6位十六进制颜色值。

6. Array.prototype·join()

a. 方法定义

join() 方法将数组中的所有元素连接成一个字符串。

b. 应用

getRandom函数使用了匿名函数 function(h) {return new Array(7 - h.length).join("0") + h}。

new Array(7-h.length)用于生成一个数组，其元素数量取决于h的长度。不同length的h可以得到不同的结果。

由上图可知：

  • 若h为6位字符串，则 new Array(7-h.length) 结果为””，匿名函数返回的结果是h本身；

  • 若h为5位字符串时，new Array(7-h.length) 结果为”0”，匿名函数返回”0” + h，即在h前补一个”0”；

  • 以此类推……

前面已知 （Math.random() * 0x1000000 << 0）.toString(16)的结果可能少于6位。应用上述匿名函数即可确保最终结果正好是6位十六进制数值。

知识延伸：

new Array(num).join(“*”) 可前置任何字符来保证结果位数为定值。

7.立即执行的匿名函数

上一节提及的匿名函数在声明后进行进行了调用：将 Math.random() * 0x1000000 << 0）.toString(16) 作为实参代入匿名函数中，这种函数可以称为立即执行的匿名函数。

立即调用的匿名函数使用方法有两种：

  1. (function(){//code}())
  2. (function(){//code})()

getRandomColor函数中使用的是第2种。“(”的作用是让解释器将括号内匿名函数解释为函数定义表达式而非函数声明语句，这样才能直接调用，否则会出现报错，详见文章最后的链接。
